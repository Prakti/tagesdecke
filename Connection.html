Class: Connection

Connection

A connection object aims to provide all RESTful HTTP verbs, supported by CouchDB via convenience functions which provide the require mechanisms for creating the proper HTTP requests.

The bare-metal HTTP request handling function returning a promise.

The HTTP verb to be used in the request

Name Type Description

Object containing key/value pairs of custom HTTP Headers
parms Object

Object containing key/value pairs of parameters to be appended to the URL must be JSON encodable

Object or Array to be JSON encoded and put into the request body
